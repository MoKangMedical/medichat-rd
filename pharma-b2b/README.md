# MediChat-RD 药企付费MVP

> **越狱思维验证 #1**：患者是用户，药企是付费方

## 商业模式

```
Parakeet Chat:  囚犯(用户) → 家属付费($15/月)
MediChat-RD:    患者(用户) → 药企付费(¥150K/年)
```

**核心逻辑**：每个罕见病患者每年为药企贡献¥15万+药品收入。依从性每提升1%，药企多赚数百万。平台年费¥15万 = 药企零头中的零头。

## 越狱思维五大映射

| 原则 | Parakeet Chat | MediChat-RD 药企版 |
|------|--------------|-------------------|
| 分离支付 | 囚犯用，家属付 | 患者用，药企付 |
| 管道寄生 | Corrlinks邮件 | 微信/短信/H5 |
| 无界面 | 纯邮件 | 微信对话，不装App |
| 封闭增长 | 监狱口碑 | 医院/患者组织渠道 |
| 情感定价 | $15/月(连接亲人) | ¥15万/年(留住患者) |

## 功能

- 📊 **仪表盘**：患者数、依从性、覆盖率、预估收入
- 👥 **患者管理**：列表/筛选/风险分级
- 📈 **依从性趋势**：30天服药率图表
- 🤖 **微信模拟**：模拟患者消息→AI自动回复
- 💰 **ROI计算器**：展示药企投资回报
- 📚 **教育内容**：预设疾病知识/用药指导

## ROI分析

假设：50个患者，依从性从65%提升到80%，药品年收入¥15万/人
- 额外收入：50 × 150,000 × 15% = **¥112.5万**
- 平台年费：**¥15万**
- **ROI = 7.5x**
