### 幻灯片4: 技术架构
```
┌─────────────────────────────────────┐
│        患者交互（微信小程序）         │
└──────────────────┬──────────────────┘
                   │
┌──────────────────▼──────────────────┐
│          AI Agent协作层              │
│  症状采集 → 基因解读 → 鉴别诊断      │
└──────────────────┬──────────────────┘
                   │
┌──────────────────▼──────────────────┐
│          知识库（OMIM/ClinVar）       │
└─────────────────────────────────────┘
```

### 演讲词：
> 我们的系统有4个核心Agent：
> 
> 1. **症状采集Agent**：像一位耐心的医生，详细询问症状
> 2. **基因解读Agent**：分析基因检测报告，识别致病突变
> 3. **鉴别诊断Agent**：综合症状和基因，进行罕见病筛查
> 4. **方案推荐Agent**：推荐治疗方案和临床试验
> 
> 它们协同工作，模拟真实的**多学科会诊（MDT）**流程。

**Step 1: 患者输入症状**
```
患者：医生您好，我孩子5岁了，肚子越来越大，
     脾脏肿大，经常骨痛，血小板减少
```

**Step 2: AI医生回复**
```
陈雅琴医生：您好，我是陈雅琴医生...
           根据您描述的症状组合——脾脏肿大、
           骨痛、血小板减少——我们需要高度
           警惕血液系统疾病或罕见代谢病...
           
           我建议您立即挂【儿童血液科】，
           并进行以下检查：
           1. 血常规 + 外周血涂片
           2. 腹部B超
           3. 酶活性检测（排除戈谢病）
```

**Step 3: 系统自动筛查**
```
🔍 罕见病筛查结果：

1. 戈谢病（95%匹配）
   - 关键症状：脾脏肿大、骨痛、血小板减少
   - 基因：GBA
   - 诊断：酶活性检测 + 基因检测

2. 尼曼匹克病（70%匹配）
   ...

建议：优先排查戈谢病
```

**Step 4: 治疗方案**
```
💊 戈谢病治疗方案：

1. 酶替代治疗（ERT）
   - 伊米苷酶（Cerezyme）
   - 每2周静脉注射一次

2. 底物减少治疗（SRT）
   - 麦格司他（Zavesca）
   - 口服，每日3次

⚠️ 请在专业医生指导下用药
```

## Q&A 准备

### 可能的问题：

**Q1: 你们的准确率如何？**
> A: 我们的系统是辅助诊断工具，不是替代医生。我们通过多Agent协作和RAG增强，尽可能提高筛查准确率。最终诊断仍需专业医生确认。

**Q2: 数据安全如何保障？**
> A: 我们实现了HIPAA级别的数据安全措施，包括数据脱敏、加密存储、审计日志等。

**Q3: 如何获取罕见病数据？**
> A: 我们整合了OMIM、Orphanet、ClinVar等公开数据库，并计划与医院合作获取真实病例。

**Q4: 如何推广？**
> A: 我们计划开源核心代码，与罕见病组织合作推广，并通过微信小程序降低使用门槛。
