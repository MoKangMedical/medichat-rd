RESTful API

API Documentation

MediChat-RD provides 20+ RESTful API endpoints built on FastAPI. Access diagnosis, knowledge graph, disease database, and drug repurposing programmatically.

Base URL: https://api.medichat-rd.com/v1
System

Health & Status

GET /health Health check endpoint

Returns the health status of the MediChat-RD service.

Parameters

None

Example Response

{ "status": "healthy", "version": "2.0.0", "uptime": "48h 23m" }
AI Diagnosis

Diagnosis Endpoints

POST /api/v1/diagnosis Run AI diagnosis

Multi-agent collaborative diagnosis. Returns differential diagnosis ranked by confidence.

Request Body

ParameterTypeRequiredDescription
patient_inputstringRequiredPatient symptoms in natural language
session_idstringOptionalSession ID for follow-up queries

Example Request

POST /api/v1/diagnosis Content-Type: application/json { "patient_input": "8yo boy, progressive muscle weakness 2 years, Gowers sign positive", "session_id": null }

Example Response

{ "session_id": "sess_abc123", "processing_time_s": 3.2, "result": { "diagnoses": [ {"name": "Duchenne Muscular Dystrophy", "confidence": 0.89}, {"name": "Spinal Muscular Atrophy", "confidence": 0.45} ] } }
POST /api/v1/diagnosis/symptoms Extract HPO terms

Extract standardized Human Phenotype Ontology terms from free-text symptom descriptions.

Query Parameters

ParameterTypeRequiredDescription
textstringRequiredFree-text symptom description

Example Response

{ "extracted_phenotypes": [ {"hpo_id": "HP:0001252", "name": "Muscular hypotonia", "confidence": 0.95}, {"hpo_id": "HP:0000510", "name": "Ptosis", "confidence": 0.92} ] }
Knowledge Graph

Knowledge Graph Endpoints

GET /api/v1/knowledge-graph Get full graph

Retrieve the complete rare disease knowledge graph data.

ParameterTypeRequiredDescription
diseasestringOptionalFilter by disease name
depthintegerOptional (default 2)Graph traversal depth (1-4)
GET /api/v1/knowledge-graph/search Search nodes

Search knowledge graph nodes by keyword.

ParameterTypeRequiredDescription
qstringRequiredSearch query
node_typestringOptionalFilter: disease/gene/drug/phenotype
Disease Database

Rare Disease Endpoints

GET /api/v1/rare-diseases List all diseases

List all rare diseases in the database with optional filtering.

ParameterTypeRequiredDescription
categorystringOptionalFilter by disease category
limitintegerOptional (50)Max results to return
offsetintegerOptionalPagination offset
POST /api/v1/rare-diseases/search-by-symptoms Search by symptoms

Search for rare diseases matching a set of symptoms.

Example Request

{ "symptoms": ["muscle weakness", "ptosis", "dysphagia"], "age": 28, "gender": "female" }
Drug Repurposing

Drug Repurposing Endpoints

POST /api/v1/repurposing/assess Assess drug repurposing

AI-driven drug repurposing assessment with target overlap analysis and literature evidence.

Example Request

{ "drug_name": "Metformin", "disease_name": "Gaucher Disease" }
