系统

GET/health

健康检查端点

GET/api/v1/system/status

获取所有模块加载状态

知识图谱

GET/api/v1/knowledge-graph

获取罕见病知识图谱

参数类型必需说明
diseasestring可选疾病名称/缩写
depthinteger可选 (默认2)图深度 1-4
GET/api/v1/knowledge-graph/search

搜索知识图谱节点

参数类型必需说明
qstring必需搜索关键词
node_typestring可选节点类型: disease/gene/drug/phenotype

罕见病数据库

GET/api/v1/rare-diseases

列出所有罕见病

参数类型必需说明
categorystring可选分类筛选
limitinteger可选 (默认50)返回数量
offsetinteger可选偏移量
GET/api/v1/rare-diseases/{name}

获取单个罕见病详情

POST/api/v1/rare-diseases/search-by-symptoms

根据症状搜索罕见病

{ "symptoms": ["肌无力", "眼睑下垂", "吞咽困难"], "age": 28, "gender": "女" }
