## 🎬 场景五：技术亮点（3:45 - 4:30）

### 画面：代码片段 + 架构图

### 旁白：
> "让我简单介绍一下技术亮点：
>
> **第一，多Agent协作**。我们用CrewAI框架，让4个专业Agent像真实MDT一样协作。
>
> **第二，RAG增强**。对接OMIM、ClinVar等权威数据库，不是凭空回答，而是有据可依。
>
> **第三，中文优化**。针对中文语境优化，支持中文基因报告解读，符合中国医疗流程。
>
> **第四，可扩展架构**。采用FastAPI微服务设计，可以轻松接入新的知识库和Agent。"

### 画面展示关键代码片段：
```python
# 多Agent协作核心代码
from crewai import Agent, Task, Crew

symptom_agent = Agent(
    role="症状采集医生",
    goal="详细采集患者症状",
    backstory="你是急诊科陈雅琴医生..."
)

gene_agent = Agent(
    role="基因解读医生",
    goal="解读基因检测报告",
    backstory="你是内科李明辉医生..."
)

# CrewAI 自动协调多个Agent工作
crew = Crew(
    agents=[symptom_agent, gene_agent, diagnosis_agent, treatment_agent],
    tasks=[symptom_task, gene_task, diagnosis_task, treatment_task]
)
```
