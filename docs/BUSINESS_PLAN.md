## 一、执行摘要

### 1.1 使命
让罕见病不再"罕见"。通过Agentic AI技术，将罕见病平均诊断延迟从5.4年缩短至数小时。

### 1.2 核心定位
**中国版OpenEvidence × 罕见病垂直领域**
- 免费给基层医生AI诊断工具
- 只对接PubMed/Orphanet/OMIM权威数据
- 靠药企"诊断决策时刻推荐位"变现

### 1.3 技术壁垒
MediChat-RD是国内首个集成**Nature 2026 DeepRare架构**的罕见病AI平台，拥有**24大核心技术模块**：

| 层级 | 模块数 | 代表模块 |
|------|--------|---------|
| 🧠 诊断核心 | 8 | HPO提取+幻觉防护+检验解析+多Agent编排 |
| 🩺 医生助手 | 5 | 语音转写+AI警报+病历生成 |
| 💊 商业化 | 3 | OpenEvidence引擎+药企广告+CPC计费 |
| 🧬 基因组 | 1 | HPO+基因组变异联合分析 |
| 👥 患者管理 | 3 | 注册+匹配+病例管理 |
| 📚 知识层 | 3 | 知识图谱+NLP+检索 |
| 🔬 研究层 | 3 | 药物重定位+研究Agent |
| **合计** | **24** | **~16,000行代码** |

### 1.4 融资需求
- **种子轮**：500万人民币
- **投前估值**：2,000万人民币
- **用途**：产品研发40% / 数据建设30% / 合规资质20% / 运营10%
- **里程碑**：12个月内完成3家三甲医院试点，覆盖30%罕见病科室

## 四、商业模式（OpenEvidence化）

### 4.1 核心逻辑

```
参考OpenEvidence（23亿美金估值）：
├── 免费给基层医生AI诊断工具
├── 只对接顶级医学期刊（PubMed/Orphanet/OMIM）
└── 药企在"诊断决策时刻"付费推荐
```

### 4.2 免费层策略

| 免费功能 | 目标用户 | 月使用上限 |
|----------|---------|-----------|
| DeepRare诊断 | 基层医院医生 | 1,000次 |
| 医生AI助手 | 罕见病科室 | 1,000次 |
| 症状自查 | 所有用户 | 不限 |
| 知识库检索 | 所有用户 | 不限 |

### 4.3 药企变现模式

| 决策时刻 | 触发条件 | 推荐内容 | 药企付费 |
|----------|---------|---------|---------|
| 鉴别诊断 | 医生输入症状 | 相关疾病+检测推荐 | ¥5-10/点击 |
| 开检查单 | 医生选择检测 | 具体检测产品 | ¥5-20/点击 |
| 制定方案 | 医生选择治疗 | 具体药物推荐 | ¥8-20/点击 |
| 临床试验 | 确诊后 | 相关临床试验 | ¥3-10/点击 |
