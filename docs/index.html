⚡ 快速开始

git clone https://github.com/MoKangMedical/medichat-rd.git
cd medichat-rd
pip install -r requirements.txt

# 启动服务器
python server.py

# 或使用 uvicorn
uvicorn server:app --host 0.0.0.0 --port 8000 --reload

# 访问 API 文档
open http://localhost:8000/docs
