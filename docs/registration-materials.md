## 三、项目概述

### 3.1 问题背景

中国罕见病患者面临的困境：
- 📊 **2000万**罕见病患者
- ⏱️ 平均确诊时间 **4.3年**
- 👨‍⚕️ 平均看 **7-8位**医生才能确诊
- ❌ **50%以上**曾被误诊
- 💔 很多患者"确诊即晚期"

### 3.2 解决方案

**MediChat-RD**：模拟真实MDT（多学科会诊）的AI诊断平台

```
患者输入症状 → 症状采集Agent → 基因解读Agent → 鉴别诊断Agent → 方案推荐Agent
                    ↓                  ↓                ↓               ↓
              结构化SOAP报告    致病突变识别      罕见病可能性排序    治疗方案推荐
```

### 3.3 核心功能

1. **智能症状采集**：多轮对话，自动识别罕见病特征症状
2. **基因报告解读**：关联ClinVar/OMIM数据库，识别致病突变
3. **鉴别诊断**：基于症状+基因的多维度分析，输出罕见病匹配度
4. **方案推荐**：治疗方案、临床试验匹配、用药指导

## 四、技术方案

### 4.1 技术栈

| 层次 | 技术选型 |
|------|---------|
| AI框架 | CrewAI（多Agent协作） |
| LLM | 小米MIMO v2-pro |
| 后端 | Python 3.12 + FastAPI |
| 前端 | React + TailwindCSS |
| 数据库 | PostgreSQL + Milvus（向量检索） |
| 知识库 | OMIM + ClinVar + Orphanet |
| 部署 | Docker + 云服务 |

### 4.2 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                   患者交互层                              │
│              Web端 / 微信小程序                           │
└───────────────────────┬─────────────────────────────────┘
                        │
┌───────────────────────▼─────────────────────────────────┐
│                   AI Agent协作层                         │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐ │
│  │ 症状采集  │→│ 基因解读  │→│ 鉴别诊断  │→│ 方案推荐  │ │
│  │  Agent   │  │  Agent   │  │  Agent   │  │  Agent   │ │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘ │
└───────────────────────┬─────────────────────────────────┘
                        │
┌───────────────────────▼─────────────────────────────────┐
│                   知识与数据层                            │
│  ┌────────────┐  ┌────────────┐  ┌────────────┐         │
│  │ 罕见病知识库 │  │ 基因数据库  │  │ 临床指南库  │         │
│  │  (OMIM)    │  │  (ClinVar) │  │ (UpToDate) │         │
│  └────────────┘  └────────────┘  └────────────┘         │
└─────────────────────────────────────────────────────────┘
```

### 4.3 关键技术创新

1. **多Agent协作诊断**：不是单一AI，而是"AI医疗团队"，模拟真实MDT流程
2. **RAG增强知识库**：整合OMIM/ClinVar/Orphanet权威数据库，减少幻觉
3. **中文罕见病优化**：针对中文语境优化，支持中文基因报告解读
4. **医生形象系统**：6位虚拟医生形象，增加患者信任感
