### 团队信息
- **项目名称**: MediChat-RD（罕见病AI辅助诊断平台）
- **参赛赛道**: 罕见病基因AI诊断
- **团队角色**: AI开发者 + 临床医生

---

## 一、项目背景

### 1.1 问题陈述
- 中国约有**2000万罕见病患者**
- 平均诊断时间长达**4.3年**（"诊断长征"）
- 患者平均需要看**7-8位医生**才能确诊
- **50%以上**的罕见病患者曾被误诊

### 1.2 核心目标
> 将罕见病诊断等待时间从**4.3年**压缩至**48小时**

### 1.3 解决方案
**MediChat-RD**: 基于多Agent协作的罕见病AI辅助诊断平台

---

### 2.2 核心Agent设计

#### Agent 1: 症状采集Agent（陈雅琴医生形象）
- 多轮对话采集详细症状
- 结构化输出：SOAP格式
- 支持罕见病特征识别

#### Agent 2: 基因解读Agent（李明辉医生形象）
- 解读基因检测报告
- 识别致病突变
- 关联OMIM/ClinVar数据库

#### Agent 3: 鉴别诊断Agent（王建国医生形象）
- 基于症状+基因的多维度分析
- 罕见病可能性排序
- 引用临床指南和文献

#### Agent 4: 方案推荐Agent（赵晓燕药师形象）
- 治疗方案推荐
- 药物信息查询
- 临床试验匹配

## 三、技术创新点

### 3.1 多Agent协作诊断
- 不同Agent负责不同诊断维度
- 模拟真实MDT（多学科会诊）流程
- 提高诊断准确率

### 3.2 RAG增强知识库
- 整合OMIM、Orphanet、ClinVar等权威数据库
- 实时检索最新医学文献
- 减少AI幻觉

### 3.3 中文罕见病优化
- 针对中文语境优化
- 符合中国医疗流程
- 支持中文基因报告解读

### 3.4 患者友好设计
- 多医生形象，增加信任感
- 通俗易懂的解释
- 支持家属协助问诊
